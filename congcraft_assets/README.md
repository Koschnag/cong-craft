# CongCraft Asset Upgrade Pack

Dieser Ordner ist ein vollständiges Starter-/Upgrade-Pack für CongCraft.

Enthalten:
- OBJ-Modelle als Platzhalter / Austauschziele
- PNG-Dateien für UI/HUD, Icons, Portraits und Decals
- WAV-Dateien für Musik und SFX
- Manifest, Style-Guide, Prompt-Dateien und Import-Contract

Wichtig:
- Diese Assets sind bewusst leichtgewichtig und generiert.
- Sie sind als Drop-in-Starterpack gedacht, nicht als finaler Art-Pass.
- Deine prozeduralen Fallbacks bleiben die Hauptlogik.

Empfohlene erste Upgrades:
1. Spieler
2. Bandit
3. Skeleton
4. Boss
5. Waffen
6. UI-Icons
7. Bäume / Ruinen
