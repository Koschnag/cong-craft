# Asset Import Contract

## Modelle
- Format: OBJ
- Up-Axis: Y
- Maßstab: 1 Einheit = 1 Meter
- Pivot: am Boden zentriert, wenn sinnvoll
- gleiche Dateinamen für Drop-in-Replacement
- *_col.obj nur vereinfachte Kollisionsgeometrie

## UI
- PNG mit Alpha
- auf dunklem Hintergrund lesbar

## Audio
- WAV 44.1 kHz
- kurze SFX mit sauberem Anfang/Ende
- Musik idealerweise loopbar

## Stil
- Mid-2000s Euro-RPG
- lieber rau / glaubwürdig als zu sauber
